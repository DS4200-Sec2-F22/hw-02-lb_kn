Data Types

UFO

  • row data type:
    • UFO sighting > item
  • column data types:
    • Datetime > attribute (ordered quantitative)
    • City > attribute (categorical)
    • State > attribute (categorical)
    • Country > attribute (categorical)
    • Shape > attribute (categorical)
    • Duration > attribute (ordered quantitative)

Airports

  • row data type:
    • Airport > item
  • column data type:
    • AIRPORT_NAME > attribute (categorical)
    • LATITUDE > position
    • LONGITUDE > position

Google-Web

  • row data type:
    • nodes
  • column data type:
    • FromNodeId > link
    • ToNodeId > link
